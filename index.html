Coding project :

MADAM is a palindrome. A palindrome is a word, phrase, number, or other sequence of characters that reads the same forward and backward, disregarding spaces, punctuation, and capitalization. Essentially, it's a sequence that remains unchanged when read in reverse order. In this app, on the next page, you can check whether your phrase is a Palindrome or not. And on the last page, you can find the details of the code.

The code :

First, I get the input element by using getElementById method to get and check any phrase or value that you put into the search area inside my checkPalindrome() function. I initially added an “invisible” class to remove any alert styling from the same result HTML. I get the value inside the search area and create a new variable “checkInput” to lowercase the letter and remove any punctuation, special characters, or spaces by using “toLowerCase()” and “replace()” methods. replace() method is used twice: The first replace(), replaces all punctuation, special characters and underscores with an empty string. The second replace(), removes all extra spaces by replacing one or more consecutive spaces with a single space. Then I reverse the checkInput variable by using “split()”, “reverse() and “join()” methods. Then by using the if-else method, I check if the reversed phrase is equal to the original phrase or not, if so by clicking on the check button on your screen you will see the result which says your entry is a palindrome, otherwise is not.
